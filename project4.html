Multimodal Deep learning for Visual Question Answering

Visual question answering is a deep learning task where an image and an associated question are passed to the model, and the model must answer the question based on the information in the image. VQA presents a difficult task for deep learning models, as they must implicitly understand the information in an image. Transformer based networks have proven up to this task and perform well to answer questions relating to information in images.

We built a transformer based model that answered questions with respect to information present in images. The dataset that we used to implement this task was called Neural Symbolic VQA, and was a state of the art dataset to implement the model.

We reached a maximum accuracy of 70% on the dataset, although we were only able to train it partially due to time constraints.
